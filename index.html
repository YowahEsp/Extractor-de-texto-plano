🚀 Extractor de Texto Automático

Extrae texto de páginas web de forma automática

📌 Método 1: Bookmarklet (Recomendado)

Arrastra este botón a tu barra de marcadores. Cuando hagas clic en él, extraerá el texto de la página actual automáticamente.

🚀 Extraer Texto
Cómo usar:
  1. Arrastra el botón "🚀 Extraer Texto" a tu barra de marcadores
  2. Abre cualquier página web
  3. Haz clic en el marcador
  4. ¡El texto se copia automáticamente y aparece una notificación!

💻 Método 2: Script de Consola

Copia y pega este código en la consola del navegador (F12 → Console)

(function(){let t=document.body.innerText;let c=t.length;let w=t.split(/\s+/).filter(p=>p).length;console.clear();console.log('%c✅ TEXTO EXTRAÍDO','font-size:20px;font-weight:bold;color:#10b981;background:#d1fae5;padding:10px');console.log(`\n📊 ${c.toLocaleString()} caracteres | ${w.toLocaleString()} palabras\n`);navigator.clipboard.writeText(t).then(()=>{console.log('%c📋 ¡Copiado al portapapeles!','font-size:16px;color:#10b981;font-weight:bold')});window.textoExtraido=t;window.descargarTexto=function(){let b=new Blob([t],{type:'text/plain'});let u=URL.createObjectURL(b);let a=document.createElement('a');a.href=u;a.download='texto-'+Date.now()+'.txt';a.click();URL.revokeObjectURL(u);console.log('💾 Descargado!')};console.log('\n💡 Comandos disponibles:');console.log('  • descargarTexto() - Descarga como TXT');console.log('  • copy(textoExtraido) - Volver a copiar')})();
Cómo usar:
  1. Abre la página web que quieres extraer
  2. Presiona F12 (o Ctrl+Shift+I / Cmd+Option+I)
  3. Ve a la pestaña "Console"
  4. Haz clic en "📋 Copiar" arriba
  5. Pega en la consola y presiona Enter

🌐 Método 3: Extraer desde URL (Experimental)

Ingresa una URL para intentar extraer su texto. Nota: Solo funciona con sitios que permitan ser cargados en iframe.

⚠️ Limitación: Muchos sitios web bloquean ser cargados en iframes por seguridad. Si no funciona, usa el Método 1 o 2.
