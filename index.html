Extractor de texto /

Captura el texto de cualquier página web y lo entrega listo para copiar o descargar, en texto plano o Markdown. Todo ocurre en tu navegador: no se envía nada a ningún servidor.

1 Marcador (recomendado)

Arrastra este botón a tu barra de marcadores. Al pulsarlo en cualquier web, abre un panel flotante con el texto extraído y opciones de ámbito y formato.

Extraer texto
Arrástralo a la barra de marcadores ↑
ÁmbitoContenido principal, página completa o solo lo seleccionado.
FormatoTexto plano o Markdown (títulos, listas, enlaces, tablas).
Panel aisladoShadow DOM: el CSS de la web no lo deforma. Se puede arrastrar.
ExportarCopiar al portapapeles o descargar con nombre y fecha automáticos.
  1. Arrastra Extraer texto a la barra de marcadores.
  2. Abre la web que quieras y, si te interesa, selecciona un fragmento.
  3. Pulsa el marcador: ajusta ámbito/formato y copia o descarga.

2 Consola del navegador

Misma lógica, por si tu navegador restringe los marcadores. Copia el código, abre la consola (F12 → Console), pégalo y pulsa Enter.
